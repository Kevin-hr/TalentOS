# 个人技能产品化规格书 (Skill-to-Product Spec)

> **版本**：0.2 (Draft by Assistant)
> **目的**：定义一款“帮助职场人将个人技能转化为标准化产品”的 AI 工具。

## 1. 核心价值定义 (Value Proposition)

*   **我是谁 (Expertise)**：拥有“行业SOP产品化”方法论的转型专家。
*   **我要解决什么问题 (Problem)**：
    *   许多职场人（尤其是35岁+）拥有丰富的行业经验（如HR招聘、财务审计、供应链管理），但不知道如何将其转化为可售卖的标准品。
    *   他们往往停留在“出卖时间”（咨询/打工）的阶段，无法实现“睡后收入”。
*   **我的解决方案 (Solution)**：**Skill-to-Product (S2P) 智能顾问**。
    *   通过深度问答挖掘用户的隐性知识。
    *   利用 AI 自动匹配市场需求，生成 MVP 产品方案。
*   **一句话产品介绍**：帮你把“脑子里的经验”变成“能卖钱的产品”的 AI 创业导师。

## 2. SOP 逻辑解构 (The "Algorithm")

这是核心业务逻辑：**简历诊断 -> 技能挖掘 -> 优先级排序**。

### 2.1 输入 (Input)
*前置条件：必须基于真实履历和意愿进行分析*

1.  **用户简历 (Mandatory)**：上传 PDF/Word 或粘贴文本。这是所有分析的基石。
2.  **个人意愿偏好 (Preferences)**：
    *   **擅长领域**：你觉得自己在哪里最自信？
    *   **激情所在**：哪类工作让你进入“心流”状态？（排除那些“虽然我会但很讨厌”的工作）
    *   **变现目标**：期望的收入模式（一次性付费/订阅制/高客单咨询）。

### 2.2 处理逻辑 (Process/SOP)
*AI 内部处理流程：*

1.  **第一步：HRBP 深度阅卷 (HRBP Diagnostic)**
    *   *角色设定*：扮演一位拥有15年经验的资深 HRBP。
    *   *动作*：扫描简历，识别那些“市场价值高”但用户可能“习以为常”的隐性技能（例如：把“负责会议记录”识别为“高效信息结构化能力”）。
    *   *产出*：一份**《潜在技能清单》**。

2.  **第二步：意愿交叉验证 (Passion/Skill Alignment)**
    *   将《潜在技能清单》与用户的《个人意愿偏好》进行比对。
    *   *过滤*：剔除“高技能低意愿”的项目（避免把用户推向他讨厌的坑）。
    *   *保留*：筛选出“高技能+高意愿”的 **甜蜜点 (Sweet Spot)**。

3.  **第三步：市场价值锚定与排序 (Market Value & Prioritization)**
    *   根据市场稀缺性、变现难易度、交付成本三个维度，对甜蜜点技能进行打分。
    *   生成 **P0-P5 优先级列表**。

### 2.3 输出 (Output)
*用户最终拿到什么？*

一份 **《技能变现战略地图》**，核心包含 **P0-P5 优先级排序**：

*   **P0 (立刻做 / Must Do)**：
    *   *定义*：技能极强 + 意愿极高 + 市场刚需（变现路径最短）。
    *   *行动*：直接生成产品 MVP 方案（如：SaaS 工具逻辑、课程大纲）。
*   **P1 (规划做 / Should Do)**：
    *   *定义*：市场价值高，但目前封装程度不够，需要整理素材。
    *   *行动*：列出需要补充的文档或案例清单。
*   **P2 (尝试做 / Could Do)**：
    *   *定义*：竞争激烈但有个人特色，适合作为引流款。
*   **P3-P5 (暂缓 / Won't Do)**：
    *   *定义*：低价值劳动或用户厌恶的工作，建议外包或舍弃。

此外，包含每个 P0/P1 项目的**具体落地建议**（SaaS/课程/咨询）。

## 3. 目标用户画像 (Target Audience)

*   **核心人群**：[B. 职场突围] - 拥有“一技之长”但陷入职业瓶颈的中年职场人。
*   **付费意愿**：**高**。他们急需转型，且本身有积蓄，愿意为“指路明灯”付费。
*   **使用频率**：**低频**（通常只规划一次），但**深度大**（涉及职业生涯重构）。

## 4. 竞争优势 (Why You?)

*   **现有替代方案**：
    *   **职业规划师**：收费贵（几千/小时），且只给建议不给产品方案。
    *   **知识付费课程**：“教你做副业”的课，往往只教理论，不落地。
*   **你的独特优势**：
    *   **结果导向**：直接生成**可执行的产品方案**，甚至直接生成代码（结合秒哒的能力）。
    *   **低成本试错**：AI 咨询成本极低，可以反复生成不同方向的方案。
