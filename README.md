## ⚡ 3. 快速开始 (Quick Start)

### 后端 (Backend)
```bash
cd TalentOS
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn src.api_server:app --reload
```

### 前端 (Frontend)
```bash
cd frontend-web
pnpm install
pnpm dev
```
