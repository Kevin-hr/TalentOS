# 阿里云服务器购买避坑指南 (省时版)

为了让你的小程序能尽快上线，且避免漫长的备案流程（国内服务器必须备案，耗时 20 天+），请务必按照以下建议购买：

## 1. 核心配置 (照着选)

*   **地域 (Region)**: 强烈建议选择 **中国香港 (China Hong Kong)** 或 **新加坡**。
    *   *原因*: **免备案**。买完就能立刻部署上线。如果选杭州/北京，你的域名必须先备案才能用，非常麻烦。
*   **操作系统 (Image)**: 选择 **Ubuntu 22.04 64位** 或 **Debian 12**。
    *   *原因*: 相比 Windows Server，Linux 更稳定、更安全、跑 Python 更快，且不仅省钱（Windows 需授权费）还省内存。
*   **网络 (Network)**: 必须勾选 **“分配公网 IPv4 地址”**。
    *   *带宽*: 推荐选择 **按使用流量付费** (带宽拉到 100M)，这样初期成本极低，速度还快。或者选固定带宽 3M 以上。
*   **安全组 (Security Group)**: 确保勾选开通 **80, 443, 22** 端口。

## 2. 购买后的操作

1.  **设置密码**: 购买时或购买后，设置一个 `root` 用户的登录密码，并**一定要记下来**。
2.  **获取 IP**: 购买成功后，回到实例列表，找到那个 **公网 IP**。
3.  **填入 Cloudflare**:
    *   回到 Cloudflare DNS 设置。
    *   找到 `bmwuv.com` 的 **A 记录**。
    *   把 IP 地址改成你新买的这个公网 IP。
    *   保存。
