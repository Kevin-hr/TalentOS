# WeChat Mini Program Frontend

This directory will contain the native WeChat Mini Program source code.

## Architecture
- **Framework**: Native WeChat Mini Program
- **Language**: TypeScript
- **UI Library**: Vant Weapp
- **API**: Connects to `https://bmwuv.com/analyze`

## Development
1. Open WeChat Developer Tools.
2. Import this directory.
3. Set AppID.
